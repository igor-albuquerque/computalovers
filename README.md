## Descrição do Projeto

O projeto tem o objetivo de fazer a operação automatizada de um braço robótico (Robot Arm H25). O sistema foi programado para realizar tarefas de coleta e posicionamento através de 3 mecanismos:

1. **Rotação da Base:** Permite o deslocamento horizontal do robô para alcançar determinados ângulos de operação.
2. **Elevação e Declinação (Cotovelo):** Controla a altura do braço, permitindo que ele erga objetos.
3. **Fechamento e Abertura da Garra:** Responsável por abrir e fechar, garantindo a captura do objeto

**O código usa a versão 2.0.0 do Pybricks**
